# [Renderer]({{site.url}}/Pages/Reference/Renderer.html).SkyLight

## Description
Sets the lighting information for the scene! You can build one through
`SphericalHarmonics.FromLights`, or grab one from `Tex.FromEquirectangular` or
`Tex.GenCubemap`
