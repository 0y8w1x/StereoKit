# [U]({{site.url}}/Pages/Reference/U.html).mm

## Description
Converts millimeters to meters. There are 1000mm in 1m. In StereoKit
1 unit is 1 meter, so `250 * Units.mm2m == 0.25`, 250 millimeters is .25
meters/units.
